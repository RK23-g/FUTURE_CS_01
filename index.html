Audit Vulnerability Findings

High
Missing Content Security Policy (CSP) Header
Web Infrastructure Security
▼
Vulnerability ID CWE-693 (Protection Mechanism Failure)
Business Risk If an attacker finds an input injection point, they can load malicious scripts (XSS), steal session cookies, or capture keystrokes.
Technical Observation The HTTP response header Content-Security-Policy is completely missing. Browsers will execute any script hosted on any domain.
Evidence

Raw Headers Received:

HTTP/1.1 200 OK Server: nginx/1.19.0 (Ubuntu) Content-Type: text/html; charset=UTF-8 (No Content-Security-Policy header present)
🛡️ Practical Remediation Steps
# For Nginx: add_header Content-Security-Policy "default-src 'self'; script-src 'self'; style-src 'self' 'unsafe-inline';";
High
Outdated PHP Engine Version Disclosure
Software Patch Management
▼
Vulnerability ID CWE-937 (Outdated Component / Version Disclosure)
Business Risk Using end-of-life (EOL) software means the server has publicly known vulnerabilities that will never be patched by developers.
Technical Observation The X-Powered-By header leaks that the site runs PHP 5.6.40. PHP 5.6 reached End-of-Life in December 2018.
Evidence
X-Powered-By: PHP/5.6.40-38+ubuntu20.04.1+deb.sury.org+1
🛡️ Practical Remediation Steps
# php.ini configuration: expose_php = Off
Medium
Missing X-Frame-Options Header (Clickjacking Risk)
Web Infrastructure Security
▼
Vulnerability ID CWE-1021 (Frame Overlay Clickjacking)
Business Risk Attackers can embed this website inside an invisible iframe to hijack user mouse clicks.
Evidence

Header check: No X-Frame-Options or frame-ancestors directive in headers.

🛡️ Practical Remediation Steps
# For Nginx: add_header X-Frame-Options "SAMEORIGIN";

Insecure Session Cookie Configuration
Session Management

Set-Cookie: testphp_session=deleted; expires=Thu, 01-Jan-1970 00:00:01 GMT; Max-Age=0; path=/ (Missing: HttpOnly, Secure, SameSite flags)
🛡️ Practical Remediation Steps
session.cookie_httponly = True session.cookie_secure = True session.cookie_samesite = "Lax"
